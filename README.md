# Dimensionality Reduction Project

## Project Overview
This project explores dimensionality reduction techniques using
PCA and SVD on real-world datasets including handwritten digits,
biometric face images, and the Iris dataset.

## Methods
- Principal Component Analysis (PCA)
- Singular Value Decomposition (SVD)
- Low-rank matrix approximation

## Key Insights
- PCA reveals latent clustering structures not visible in raw space
- Increasing rank improves reconstruction but shows diminishing returns
- Some datasets (e.g. Iris) are not linearly separable in low dimensions

## Tools
- MATLAB
- Linear Algebra
- Data Visualization
